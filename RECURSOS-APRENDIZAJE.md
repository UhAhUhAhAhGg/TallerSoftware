## 🏆 Proyectos de Ejemplo para Estudiar

### Pequeños (7-10 horas)
1. **Lista de Tareas**
   - Backend: GET todos, POST crear, DELETE eliminar
   - Frontend: Mostrar lista, agregar, eliminar
   - Base de datos: JSON file (sin SQL por ahora)

2. **Weather App**
   - Backend: Proxy API pública
   - Frontend: Buscar ciudad, mostrar clima
   - API: openweathermap.org (gratuita)

### Medianos (20-30 horas)
1. **Blog Simple**
   - Backend: CRUD de posts
   - Frontend: Crear, editar, eliminar posts
   - Base de datos: MongoDB (atlas gratuito)

2. **E-commerce Básico**
   - Backend: Productos, carrito, órdenes
   - Frontend: Catálogo, carrito, checkout
   - Base de datos: MongoDB

## 💡 Consejo Final

**No intentes aprenderlo todo a la vez.** Sigue este orden:
1. Express básico → Entiende peticiones HTTP
2. React básico → Entiende componentes
3. Conectar ambos → Entiende la comunicación
4. Hacer algo funcional → Puedes agregar más

Es como aprender a manejar: primero aprendes lo básico, luego cambios, luego estacionamiento, etc.

¡Éxito en tu aprendizaje! 🎉
